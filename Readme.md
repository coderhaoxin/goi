
### goi :dancers:

`goi == go get (binary only, no repo)`

Only want to get the binary ? Such as some command line tools.

Oh, this is for you. :dancer:

### Usage

```bash
$ go get github.com/coderhaoxin/goi

# install something
$ goi github.com/rakyll/boom

# or
$ goi rakyll/boom

# with flags
$ goi -u rakyll/boom
```
